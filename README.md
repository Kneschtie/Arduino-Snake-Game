# Arduino-Snake-Game
Playing snake on a arduino

# What you need:

_-Arduino Uno
_-Joystick
_-8x8 MAX7219 DotMatrix-Module
_-Breadbord (optional)

# How to connect:

## Joystick:
  GND - GND
  +5v - 5v
  JRx - A0
  JRy - A1
  SW  - Pin9
  
## Matrix-Module
  VCC - 5v
  GND - GND
  DIN - Pin12
  cs  - Pin10
  CLK - Pin11
